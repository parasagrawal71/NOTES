<pre>

    │	    ├── PageName.css

    │       └── PageName.scss
    │
    ├── services/apis
    │   ├──	handleError.js
    │   ├── httpConstants.js
    │   └── apiProviderName.js
    │
    ├── redux

    │	│   └── types.js

__Some Points to keep in mind:-__
 * Capitalize Component names, Page names and App files.   
 * Function names should start with lowercase letter.
